### Objectives
- Complete 5 problems:
    1. Binary Words and Operations
        - Insert functions in python: parity, ch, maj and four sigma functions.
    2. Fractional Parts of Cube Roots
        - Calculate constants of the Secure Hash Standard and compare.
    3. Padding
        - Process messages according to section 5.1.1 and 5.2.1 of the Secure Hash Standard.
    4. Hashes
        - Write a function that calculates the next hash value, following section 6.2.2 of the SHA-256 Hash Computation
    5. Passwords
        - Determine the three given passwords by 'attacking' them and explain how to imporve and prevent the attack.
